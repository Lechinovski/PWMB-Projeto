Descrição do Curso

O CSS é uma linguagem usada para controlar a apresentação visual de páginas da web escritas em HTML.

Neste curso, você aprenderá como usar o CSS para definir como os elementos HTML devem ser exibidos na tela, abrangendo cores, fontes, margens, tamanhos e posicionamento.
